# API endpoints

| HTTP Request | Endpoint                   | Action                     |
|--------------|----------------------------|----------------------------|
| POST         | /api/users/register        | Register new user          |
| POST         | /api/users/login           | Login existing user        |
| POST         | /api/users/logout          | Logout logged in user      |
| GET          | /api/listings              | List existing listings     |
| GET          | /api/listings/<listing_id> | Get details for listing_id |
