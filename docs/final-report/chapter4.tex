This project successfully developed and implemented a smart monitoring system tailored for short-term rental properties, 
specifically for AirBnB listings. By integrating sensors to monitor energy consumption, water usage, and indoor 
temperature, the system provided real-time feedback to both guests and hosts, helping promote more sustainable resource 
use.

\section{Achievement of Objectives}

The project achieved its primary objectives:

\begin{enumerate}
  \item \textbf{Smart sensors deployment:} Energy, water, and temperature sensors were deployed in a real AirBnB 
  property, effectively capturing and transmitting consumption data.
  \item \textbf{API development:} A robust REST API, adhering to OpenAPI standards, was built to allow secure access to 
  sensor data for applications and users.
  \item \textbf{Report generation:} The system generated daily and end-of-stay reports for guests, offering insights 
  into their resource consumption and encouraging eco-conscious behavior during their stays.
  \item \textbf{User-friendly web application:} A web application was created for hosts, providing them with a clear 
  interface to monitor and manage their property's resource consumption, ensuring operational efficiency while fostering 
  sustainability.
\end{enumerate}

\section{Contributions and Impact}

This project can significantly contribute to various areas, including behavioral economics by motivating guests to adopt 
sustainable practices through real-time feedback; measuring the environmental footprint of lodging and tourism; and 
establishing awards and incentives for users who demonstrate resource conservation. Additionally, it supports the 
greening of the shared economy by encouraging eco-friendly practices among hosts and guests. The implementation of 
certification programs for sustainable properties can enhance marketability, while providing demonstrable, transparent, 
and accurate measurements of resource consumption fosters trust and accountability within the hospitality sector.

\section{Recommendations for Future Work}

While the project successfully met its objectives, several areas can be explored to improve system performance and 
scalability:

\begin{enumerate}
  \item \textbf{Memory safe microcontroller programming:} Rewriting the microcontroller firmware using Rust, a 
  memory-safe programming language, would enhance system reliability, reduce the likelihood of bugs, and improve 
  overall performance.
  \item \textbf{Improved sensors accuracy:} Higher precision sensors, particularly for water flow measurement, would 
  improve the system's accuracy, especially in capturing low-flow data.
  \item \textbf{Expanded platform integration:} Integrating with other platforms like VRBO or Booking.com would extend 
  the system's utility, allowing more hosts to adopt sustainable monitoring practices.
  \item \textbf{Machine learning for appliance identification:} Incorporating machine learning models to identify 
  appliances based on their consumption patterns could enhance resource management by pinpointing specific areas of 
  improvement for hosts.
  \item \textbf{Satellite internet connectivity:} A more reliable internet connection, such as satellite internet with
  a battery backup, would ensure continuous data transmission, even in remote locations with unreliable network 
  coverage.
\end{enumerate}
    
\section{Final Thoughts}

In conclusion, this project has successfully demonstrated the potential for smart monitoring systems to drive 
sustainability in the hospitality sector. By providing real-time feedback on resource consumption, it encourages more 
responsible behavior from both guests and hosts. The project has laid a solid foundation for future enhancements and 
scaling, paving the way for further innovation in sustainable tourism. The lessons learned here emphasize the critical 
role of technology in fostering environmental awareness and creating more efficient, eco-friendly accommodation 
experiences.
