\chapter{Aim and Objectives}

The main aim of this project is to develop a data-driven solution that educates tourists about the carbon footprint of Airbnb stays, empowers hosts to optimize resource usage, and 
promotes sustainable practices in the hospitality industry.

\subsubsection{Objectives}

\begin{itemize}
    \item Create a robust system to collect data from room and outside temperature, water consumption, and energy usage in Airbnb accommodations.
    \item Prepare, process and analize the collected data in a way that can be used to calculate the total resource usage.
    \item Calculate an average score for resources usage per guest per stay per stay based on average score from previous guests.
    \item Generate a report for tourists with personalized resource usage information that can help them acknowledge the impact of their stay.
    \item Develop an intuitive dashboard for hosts to monitor and optimize resource usage.
\end{itemize}
